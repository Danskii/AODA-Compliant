Part 5: Marking Artifacts

When an element is tagged as an Artifact in Acrobat a screen reader will ignore it.

The artifact tag should be used on images that are purely decorative and do not convey any information.

Here are some, but definately not all, examples of elements which are purely decorative:

  • borders
  • section divisions
  • abstract patterns
  • crop marks

In the examples above the best approach would be to mark the image as an Artifact.

Here's how:

  1. Click on the Accessibility shortcut in the right hand panel
  2. Click on Reading Order
  3. Expand the Tags Tree
  4. Navigate within the Tags Tree to the Figure Tag you would like to mark as an Artifact
  5. Highight and double click the figure tag and type Artifact. This changes the Tag to an Artifact.

Ok, that's it for this section.
