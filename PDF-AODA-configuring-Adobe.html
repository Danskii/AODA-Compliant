Part 1: Configuring Adobe

There are a number of tools built into Adobe Acrobat that are essential to the remediation process.

Before getting started on the actual technical process of remediation you'll want to configure Adobe so these tools are easily accessible. Doing this first will make the whole process much more efficient.

Here are the steps I follow to configure Adobe for efficient remediation work.

Adding Accessibility Tools Shortcut

  1. Select Tools on the top toolbar
  2. Click the add button below the Accessibility option

Adding Edit PDF Shortcut

  1. Select Tools on the top toolbar
  2. Click the add button below the Edit PDF option

Adding Tags Shortcut

  1. Open the PDF you will be doing remediation work on
  2. Right click on left hand Navigation Pane
  3. Select order
  4. Select tags
  5. Select content

Settings Configuration

  1. Select Edit on the top toolbar
  2. Select Preferences
  3. Select Accessibility
  4. Under Other Accessibility Options ensure the following are checked:
    • Use document structure for tab order when no explicit tab order is specified
    • Enable assistive technology support
